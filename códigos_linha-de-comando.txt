- Reativa a linha de comando, caso trave
Ctrl + C

- Monitorar alterações do scss
$ npm run gulp listen

- Verifica o status do git
$ git status

- Adiciona todos os arquivos alterados para fazer o commit
$ git add .

- Inclui um novo commit
$ git commit -am 'MENSAGEM'

- Exibe todos os commits
$ git log

- Exibe o último commit
$ git log -n 1
